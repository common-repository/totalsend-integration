Using this plugin, a WordPress blog owner will be able to link his/her TotalSend account with WordPress and start accepting email list subscriptions.


== Description ==

This WordPress plugin will let you to publish mail list subscription (and un-subscription) forms on your WordPress powered websites and link these forms with your email marketing TotalSend account.

== Frequently Asked Questions ==

[TotalSend WordPress Integration Knowledgebase](http://www.totalsend.com/knowledgebase/wordpress-integration)

== Installation ==

1. Upload plugin folder to your blog (wp-content/plugins/).
2. Activate it within the admin section of your WordPress installation.
3. Once done, go to "Settings > TotalSend WP Plugin" and enter your [TotalSend Credentials], which can be obtained from your TotalSend account under *[Settings > Integrations > TotalSend WordPress Plugin](https://app.totalsend.com/app/user/integration/wordpress/)*.
4. You can now go to "Appearance > Widget", load the widget where you need it and set it up as it loads your lists from TotalSend, select any custom fields you may have and would like the user to fill in.
5. This plugin also provides a new shortcode which you can use once item No. 4 in the list has been completed as **totalsend-subscribe-form**.
